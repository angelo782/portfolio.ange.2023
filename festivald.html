Cette application web permet de gérer l'hébergement des groupes de musique durant le festival Folklores du Monde.

Elle offre les services suivants :

  • Gérer les établissements (caractéristiques et capacités d'accueil) acceptant d'héberger les groupes de musiques
  • Consulter, réaliser ou modifier les attributs aux groupes dans les établissements
